# pwa-todolist-precache-with-webpack
pwa todolist precache with webpack

## Getting Started
```
git clone https://github.com/Ava-Chang/pwa-todolist-precache-with-webpack.git
```

```
npm install
```

```
http-server
```

```
json-server db.json
```
